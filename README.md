# Sistema de Gestão de Usuários - Python

Projeto desenvolvido como parte do portfólio para o Programa de Talentos de TI. 
O sistema simula um gerenciador de cadastros bancários com foco em segurança e integridade de dados.

## 🛠️ Tecnologias e Conceitos Aplicados:
- **Lógica de Programação:** Estruturas de repetição (`while`) e controle moderno (`match-case`).
- **Manipulação de Dados:** Uso de listas dinâmicas e métodos de ordenação/remoção (`pop`, `enumerate`).
- **Validação de Segurança:** Sanitização de strings com `.strip()`, `.title()` e filtros com `.isalpha()`.
- **UX/UI no Terminal:** Interface limpa utilizando a biblioteca `os` e `msvcrt` para interação fluida.
- **Tratamento de Erros:** Prevenção de quebras de sistema com `try/except`.

## 🚀 Como rodar o projeto:
1. Tenha o Python 3.10 ou superior instalado.
2. Execute o arquivo `.py`.
